# Recipes CRUD API

## Description
This is a REST API for managing recipes using Node.js, Express.js, and MongoDB.

It supports full CRUD operations:
- Create recipe
- Get all recipes
- Get recipe by ID
- Update recipe
- Delete recipe

## Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)
- Postman
- Render

## Project Structure
recipes-crud-app/
│
├── controllers/

## API Endpoints

POST /api/recipes  
GET /api/recipes  
GET /api/recipes/:id  
PUT /api/recipes/:id  
DELETE /api/recipes/:id  

## Deployment
